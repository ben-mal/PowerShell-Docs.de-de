# Microsoft.PowerShell.Utility-Modul

## BESCHREIBUNG

Dieser Abschnitt enthält die Hilfe Themen für die Cmdlets, die mit dem Microsoft. PowerShell. Utility-Modul installiert werden. Dieses Modul enthält Cmdlets, die die grundlegenden Features von PowerShell verwalten.

## Microsoft. PowerShell. Utility-Cmdlets

### [Add-Member](Add-Member.md)
Fügt einer Instanz eines PowerShell-Objekts benutzerdefinierte Eigenschaften und Methoden hinzu.

### [Add-Type](Add-Type.md)
Fügt einer PowerShell-Sitzung eine Microsoft .net Core-Klasse hinzu.

### [Clear-Variable](Clear-Variable.md)
Löscht den Wert einer Variablen.

### [Compare-Object](Compare-Object.md)
Vergleicht zwei Sätze von Objekten.

### [ConvertFrom-Csv](ConvertFrom-Csv.md)
Konvertiert Objekteigenschaften im durch Trennzeichen getrennten (CSV-)Format in CSV-Versionen der ursprünglichen Objekte.

### [ConvertFrom-Json](ConvertFrom-Json.md)
Konvertiert eine JSON-formatierte Zeichenfolge in ein benutzerdefiniertes Objekt oder eine Hash Tabelle.

### [ConvertFrom-Markdown](ConvertFrom-Markdown.md)
Konvertiert den Inhalt einer Zeichenfolge oder einer Datei in ein **markdowninfo** -Objekt.

### [ConvertFrom-SddlString](ConvertFrom-SddlString.md)
Konvertiert eine SDDL-Zeichenfolge in ein benutzerdefiniertes-Objekt.

### [ConvertFrom-StringData](ConvertFrom-StringData.md)
Konvertiert eine Zeichenfolge, die mindestens ein Schlüssel-Wert-Paar enthält, in eine Hashtabelle.

### [ConvertTo-Csv](ConvertTo-Csv.md)
Konvertiert .NET-Objekte in eine Reihe von CSV-Zeichen folgen (Zeichen getrennte Werte).

### [ConvertTo-Html](ConvertTo-Html.md)
Konvertiert .NET-Objekte in HTML, die in einem Webbrowser angezeigt werden kann.

### [ConvertTo-Json](ConvertTo-Json.md)
Konvertiert ein Objekt in eine JSON-formatierte Zeichenfolge.

### [ConvertTo-Xml](ConvertTo-Xml.md)
Erstellt eine XML-basierte Darstellung eines Objekts.

### [Debug-Runspace](Debug-Runspace.md)
Startet eine interaktive Debugsitzung mit einem Runspace.

### [Disable-PSBreakpoint](Disable-PSBreakpoint.md)
Deaktiviert die Haltepunkte in der aktuellen Konsole.

### [Disable-RunspaceDebug](Disable-RunspaceDebug.md)
Deaktiviert das Debuggen in einem oder mehreren Runspaces und gibt alle ausstehenden Debugger-Stopps frei.

### [Enable-PSBreakpoint](Enable-PSBreakpoint.md)
Aktiviert die Haltepunkte in der aktuellen Konsole.

### [Enable-RunspaceDebug](Enable-RunspaceDebug.md)
Aktiviert das Debuggen in Runspaces, wo ein Haltepunkt beibehalten wird, bis ein Debugger angefügt wird.

### [Export-Alias](Export-Alias.md)
Exportiert Informationen zu aktuell definierten Aliasen in eine Datei.

### [Export-Clixml](Export-Clixml.md)
Erstellt eine XML-basierte Darstellung eines Objekts oder von Objekten und speichert sie in einer Datei.

### [Export-Csv](Export-Csv.md)
Konvertiert-Objekte in eine Reihe von CSV-Zeichen folgen (Comma-Separated Value, Komma getrennte Werte) und speichert die Zeichen folgen in einer Datei.

### [Export-FormatData](Export-FormatData.md)
Speichert Formatierungsdaten aus der aktuellen Sitzung in einer Formatierungsdatei.

### [Export-PSSession](Export-PSSession.md)
Exportiert Befehle aus einer anderen Sitzung und speichert Sie in einem PowerShell-Modul.

### [Format-Custom](Format-Custom.md)
Verwendet eine benutzerdefinierte Ansicht zur Formatierung der Ausgabe.

### [Format-Hex](Format-Hex.md)
Zeigt eine Datei oder eine andere Eingabe als hexadezimal an.

### [Format-List](Format-List.md)
Formatiert die Ausgabe als eine Liste der Eigenschaften, in der jede Eigenschaft in einer neuen Zeile angezeigt wird.

### [Format-Table](Format-Table.md)
Formatiert die Ausgabe als Tabelle.

### [Format-Wide](Format-Wide.md)
Formatiert Objekte als eine große Tabelle, in der nur eine Eigenschaft pro Objekt angezeigt wird.

### [Get-Alias](Get-Alias.md)
Ruft die Aliase für die aktuelle Sitzung ab.

### [Get-Culture](Get-Culture.md)

### [Get-Date](Get-Date.md)

### [Get-Event](Get-Event.md)

### [Get-EventSubscriber](Get-EventSubscriber.md)

### [Get-FileHash](Get-FileHash.md)

### [Get-FormatData](Get-FormatData.md)
Ruft die Formatierungsdaten in der aktuellen Sitzung ab.

### [Get-Host](Get-Host.md)

### [Get-MarkdownOption](Get-MarkdownOption.md)

### [Get-Member](Get-Member.md)

### [Get-PSBreakpoint](Get-PSBreakpoint.md)

### [Get-PSCallStack](Get-PSCallStack.md)

### [Get-Runspace](Get-Runspace.md)

### [Get-TraceSource](Get-TraceSource.md)

### [Get-Unique](Get-Unique.md)

### [Get-Uptime](Get-Uptime.md)

### [Get-Variable](Get-Variable.md)
Ruft die Variablen in der aktuellen Konsole ab.

### [Get-Verb](Get-Verb.md)

### [Import-Alias](Import-Alias.md)

### [Import-Csv](Import-Csv.md)

### [Measure-Command](Measure-Command.md)

### [Measure-Object](Measure-Object.md)
